# UTSTokoBuku


### Form awal pada saat di run
#### Disini dapat dilihat bahwa form dapat menampilkan tanggal dan waktu saat ini.

![ss1](https://github.com/karinawidhia/UTSTokoBuku/blob/master/0.JPG)


### Fungsi Button Save
#### Pada metode pertama, penambahan data transaksi dengan jumlah transaksi yang dibawah 100000.
#### Sehingga harga akhirnya adalah harga normal, dengan cara perhitungan hasil kali antara jumlah dan harga per satuannya.

![ss2](https://github.com/karinawidhia/UTSTokoBuku/blob/master/2.JPG)

#### Pada metode kedua, penambahan data transaksi dengan jumlah transaksi yang diatas 100000.
#### Sehingga harga akhirnya adalah harga dengan diskon 10%, dengan cara perhitungan hasil kali antara jumlah dan harga per satuannya dikurangi diskon.

![ss3](https://github.com/karinawidhia/UTSTokoBuku/blob/master/3.JPG)


### Fungsi Button Refresh
#### Button refresh berguna untuk mereload kembali data yang ada pada tabel di database.
#### Hal ini biasanya ditampilkan pada tabel yang ada.

![ss4](https://github.com/karinawidhia/UTSTokoBuku/blob/master/5.JPG)


### Fungsi Button Clear
#### Button clear menyebabkan semua data yang sudah terisi pada text field akan ter-reset.

![ss5](https://github.com/karinawidhia/UTSTokoBuku/blob/master/4.JPG)


### Fungsi Button Print
#### Button print digunakan untuk mencetak hasil akhir yang tertera pada form.

![ss6](https://github.com/karinawidhia/UTSTokoBuku/blob/master/6.JPG)
